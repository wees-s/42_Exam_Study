🎯 Objetivo do Exercício

O arquivo fornecido pela 42 (given.c) é um esqueleto cheio de "pegadinhas" (erros intencionais de compilação e lógica) e buracos que você deve preencher. O objetivo é construir um analisador (parser) matemático que respeita precedência de *, + e parênteses ().

🔍 Caça aos Bugs no given.c

1️⃣ A Pegadinha do Include

❌ CÓDIGO DA PROVA
#include <stdio.h>
#include <malloc.h> // change this to <stdlib.h>
#include <ctype.h>
✅ CÓDIGO CORRIGIDO
#include <stdio.h>
#include <stdlib.h>
#include <ctype.h>
❌ Problema: O próprio esqueleto avisa para mudar. Usar malloc.h pode causar erro na Moulinette e warnings no gcc.

2️⃣ A Pegadinha do Protótipo Fantasma

❌ CÓDIGO DA PROVA
    struct node *r;
}   node;

node *n(**s);

node    *new_node(node n)
✅ CÓDIGO CORRIGIDO
    struct node *r;
}   node;

/* Remova node *n(**s); e coloque seus protótipos reais: */
node *parse_add(char **s);
node *parse_mult(char **s);
node *parse_base(char **s);

node    *new_node(node n)
❌ Problema: O protótipo node *n(**s); é lixo sintático colocado de propósito para quebrar a compilação e te confundir. Apague-o e declare as funções que você vai criar, pois a recursão de cima para baixo vai precisar delas.

3️⃣ O Erro Fatal na Lógica de Leitura (accept)

❌ CÓDIGO DA PROVA
int accept(char **s, char c)
{
    if (**s)
    {
        (*s)++;
        return (1);
    }
    return (0);
}
✅ CÓDIGO CORRIGIDO
int accept(char **s, char c)
{
    if (**s == c)
    {
        (*s)++;
        return (1);
    }
    return (0);
}
❌ Problema: O código original avança o ponteiro para QUALQUER caractere válido (if (**s)), engolindo a string inteira na primeira leitura. Ele não checa se o caractere é o c esperado.
✅ Solução: Coloque == c para só consumir o caractere se ele for um '+' ou '*' real.

4️⃣ Variável Não Declarada (parse_expr)

❌ CÓDIGO DA PROVA
node    *parse_expr(char *s)
{
    //...

    if (*s) 
    {
        destroy_tree(ret);
        return (NULL);
    }
    return (ret);
}
✅ CÓDIGO CORRIGIDO
node    *parse_expr(char *s)
{
    node *ret = parse_add(&s);
    if (!ret)
        return (NULL);

    if (*s) 
    {
        unexpected(*s);
        destroy_tree(ret);
        return (NULL);
    }
    return (ret);
}
❌ Problema: A variável ret é usada para dar free (destroy_tree) e retornar, mas ela não foi declarada nem instanciada.
✅ Solução: Declare o node *ret chamando a função de nível mais alto do seu parser (parse_add). Além disso, se o ponteiro não chegou no \0 (if *s), chame unexpected(*s) antes de destruir a árvore para cumprir a regra de mensagens de erro.

5️⃣ Falta de Retorno (eval_tree)

❌ CÓDIGO DA PROVA
int eval_tree(node *tree)
{
    switch (tree->type)
    {
        case ADD:
            return (eval_tree(tree->l) + eval_tree(tree->r));
        ...
        case VAL:
            return (tree->val);
    }
}
✅ CÓDIGO CORRIGIDO
int eval_tree(node *tree)
{
    switch (tree->type)
    {
        case ADD:
            return (eval_tree(tree->l) + eval_tree(tree->r));
        ...
        case VAL:
            return (tree->val);
    }
    return (0);
}
❌ Problema: Falta um return fora do switch, o que gera o warning de compilação control reaches end of non-void function na flag -Werror.
✅ Solução: Adicione return (0); no fim.

🧠 Lógica da Solução: O Código Faltante (//...)

Onde o esqueleto mostra //..., você deve criar as três funções de parser. Decore essa ordem:

1. BASE: Lida com ( ) e converte char para int subtraindo '0'.
2. MULTIPLICAÇÃO: Pega a BASE no lado esquerdo. Num while procura *. Se achar, pega a BASE no direito e vira o novo nó pai.
3. ADIÇÃO: Pega a MULTIPLICAÇÃO no lado esquerdo. Num while procura +. Se achar, pega a MULT no lado direito e vira o novo nó pai.

📝 O Bloco de Código que Você Deve Inserir

/* ================= LOGICA DA ARVORE ================= */
node *parse_base(char **s)
{
    node *res;
    node tmp;

    /* Caso 1: Parênteses (recomeça a hierarquia no topo) */
    if (accept(s, '('))
    {
        res = parse_add(s);
        if (!res) return (NULL);
        if (!expect(s, ')')) 
        { 
            destroy_tree(res); /* Evita memory leak */
            return (NULL); 
        }
        return (res);
    }
    /* Caso 2: Números */
    if (isdigit(**s))
    {
        tmp.type = VAL;
        tmp.val = **s - '0';
        res = new_node(tmp);
        (*s)++;
        return (res);
    }
    /* Caso 3: Lixo sintático */
    unexpected(**s);
    return (NULL);
}

node *parse_mult(char **s)
{
    node *left = parse_base(s);
    node tmp;

    if (!left) return (NULL);
    while (accept(s, '*'))
    {
        node *right = parse_base(s);
        if (!right) 
        { 
            destroy_tree(left); /* Limpa memória se a direita falhar */
            return (NULL); 
        }
        tmp.type = MULTI;
        tmp.l = left;
        tmp.r = right;
        left = new_node(tmp);
    }
    return (left);
}

node *parse_add(char **s)
{
    node *left = parse_mult(s);
    node tmp;

    if (!left) return (NULL);
    while (accept(s, '+'))
    {
        node *right = parse_mult(s);
        if (!right) 
        { 
            destroy_tree(left); 
            return (NULL); 
        }
        tmp.type = ADD;
        tmp.l = left;
        tmp.r = right;
        left = new_node(tmp);
    }
    return (left);
}
